## Description
This project is a simple Parking Lot Management System implemented using C++.  
It allows adding slots, parking vehicles, removing vehicles, and calculating charges.

## Features
- Add parking slots  
- Park a vehicle  
- Remove a vehicle  
- Calculate parking charges  
- Display parking status  

## How to Run
1. Compile the code:
   g++ main.cpp parking_ops.cpp slot.cpp -o parking

2. Run the program:
   ./parking
